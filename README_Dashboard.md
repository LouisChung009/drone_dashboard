# 無人機標案視覺化儀表板使用說明

本專案提供一個互動式儀表板，用於分析加拿大、美國、澳洲、日本及台灣的無人機標案資料。

## 檔案說明
- **`drone_dashboard.html`**: 視覺化儀表板檔案。請直接使用瀏覽器（Chrome, Edge 等）開啟即可觀看。
- **`generate_drone_charts.py`**: 用於生成上述 HTML 檔案的 Python 程式。
- **`db_schema.txt`**: 資料庫架構說明（供參考）。

## 功能介紹
1. **多國數據**: 頁面從上到下依序顯示加拿大、美國、澳洲、日本、台灣的標案圖表。
2. **廠商篩選**:
    - **勾選清單**: 畫面左側設有勾選清單，可針對個別國家篩選特定廠商。
    - **排序功能**: 可選擇依「得標金額 (高->低)」或「廠商名稱 (A-Z)」排序列表。
    - **批次操作**: 提供「全選」與「取消全選」按鈕，方便快速篩選。
3. **視覺優化**:
    - **水平捲軸**: 針對廠商眾多的圖表（如美國），自動啟用水平捲軸，讓您可以左右拖曳檢視所有廠商，避免擁擠。
    - **智慧標籤排列**: 中文/日文廠商名稱採直式堆疊顯示，並自動移除英文部分以保持簡潔；英文名稱則旋轉 90 度顯示。
    - **版面調整**: 已隱藏 X 軸 "Vendor" 標題，並優化整體間距。
4. **詳細資訊**: 滑鼠游標移動到長條圖的色塊上，會顯示該筆標案的「名稱」與「得標金額」。
4. **縮放與平移**: 可使用滑鼠在圖表區域進行拖曳以平移，或框選特定區域以放大檢視。

## 如何更新資料
當資料庫 (`drone_tenders.db` 或 `drone_awards.db`) 更新後，請執行以下步驟以更新儀表板：

1. 開啟終端機 (Terminal) 或命令提示字元 (CMD)。
2. 切換到程式所在目錄：
   ```bash
   cd F:\Drone_Tender_Reptile
   ```
3. 執行 Python 程式：
   ```bash
   python generate_drone_charts.py
   ```
4. 程式執行完畢後，重新開啟 `drone_dashboard.html` 即可看到最新資料。

## 分享方式
本儀表板為單一 HTML 檔案 (`drone_dashboard.html`)，您可以直接將此檔案透過 Email 或通訊軟體傳送給主管或同事，他們無需安裝任何軟體即可使用瀏覽器開啟觀看。
